----------------------------------------------------------------------------
ARCHITETTURA - COMPLETATA
----------------------------------------------------------------------------
[x] Configurazione centralizzata (config.py, Singleton via costanti).
[x] Logger condiviso con rotazione file (src/logger.py).
[x] Pipeline di ingestion end-to-end:
    [x] Crawler resiliente su diem.unisa.it / docenti.unisa.it / corsi.unisa.it
        con scoperta dinamica delle pagine docenti e corsi.
    [x] Estrazione metadati HTML (title, lang, date) prima dell'extractor.
    [x] Loader PDF integrato con dedup su URL.
    [x] Filtro lingua (drop /en/, /zh/ + drop via attributo lang).
    [x] Filtro temporale (cutoff 2020) su URL + metadata + testo iniziale,
        con corsia preferenziale per pagine time-sensitive.
    [x] Generazione context header via Ollama (llama3.2:3b) con cache,
        fallback euristico se Ollama non risponde, normalizzazione e clip.
    [x] Strategia Parent-Child indexing (parent 2000/200, child 400/50)
        con ContextHeaderTextSplitter custom che prepend dell'header.
    [x] Batching adattivo in indicizzazione (max 4000 child per batch).
    [x] Persistenza Chroma + LocalFileStore per i parent.
    [x] Dump JSON dello stato post-filtri (crawled_urls.json,
        crawled_pdfs.json).

[x] Core RAG (src/brain.py):
    [x] E5HuggingFaceEmbeddings (prefissi passage:/query:) come embedding
        condiviso ingestion/runtime.
    [x] Cross-encoder reranker (mMiniLMv2) con scoring + re-sort.
    [x] ParentDocumentRetriever con similarity_score_threshold (k=20,
        thr=0.7) -> rerank top-3.
    [x] Chain LangChain: rewrite condizionale -> retrieve -> rerank ->
        format context -> LLM.
    [x] Storia di conversazione per sessione (InMemoryChatMessageHistory).
    [x] API sincrona (`chat`) e streaming generator (`chat_stream`).
    [x] Citazione fonti dedotte dai metadata.

[x] UI Gradio (app.py):
    [x] Caricamento lazy del Chroma index (rebuild se mancante o --reindex).
    [x] gr.ChatInterface con esempi prefillati IT/EN.
    [x] Streaming token-per-token via chat_stream.

[x] Tool di debug (scripts/):
    [x] check_vector_db.py (validazione propagazione context header).
    [x] test_chunks.py (ispezione child/parent chunks).
    [x] test_retriever.py (REPL interattiva sul retriever bi-encoder).

EVALUATION FRAMEWORK - COMPLETATO
----------------------------------------------------------------------------
[x] Golden set IT (golden_set_it.json) e EN (golden_set_en.json) con
    categorie in_scope / multi_turn / out_of_scope / robustness.
[x] Runner (evaluation/tester.py) con:
    [x] Ragas 0.4.3 (Faithfulness, ResponseRelevancy, AnswerCorrectness,
        LLMContextPrecisionWithReference, LLMContextRecall, AspectCritic
        custom "coherence").
    [x] Two-phase Ragas eval (no-ref vs ref-requiring) per evitare NaN
        artificiali su rows senza reference.
    [x] Custom scope-awareness check (strict marker + LLM-judge, distingue
        rejection da knowledge-gap).
    [x] Custom robustness check multi-turno (5 tag scenari avversariali)
        con marker override per i refusal-style tag.
    [x] Cache su disco (evaluation/cache.py) configurabile off/use/refresh,
        keyed per (chat_model, temperature, history pairs, question).
    [x] Output strutturati per run: run.log, per_question.json,
        ragas_metrics.csv/.json, scope_awareness.json, robustness.json,
        summary.md.
[x] Run smoke test (--limit 3) completata su lang=it:
    evaluation/results/20260511_172307_it/
